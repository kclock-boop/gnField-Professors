# 관리자 운영 매뉴얼

대한민국산업현장교수 경남협의회 홈페이지의 간단 로그인 운영용 문서입니다.

## 현재 로그인 계정 구조

| 성함 | 아이디 | 비밀번호 |
|---|---|---|
| 교수님 1 | gn01 | 1001 |
| 교수님 2 | gn02 | 1002 |
| 교수님 3 | gn03 | 1003 |
| 교수님 4 | gn04 | 1004 |
| 교수님 5 | gn05 | 1005 |
| 교수님 6 | gn06 | 1006 |
| 교수님 7 | gn07 | 1007 |
| 교수님 8 | gn08 | 1008 |
| 교수님 9 | gn09 | 1009 |
| 교수님 10 | gn10 | 1010 |

위 표의 `성함` 칸은 실제 교수님 성함으로 바꿔서 관리하면 됩니다.

## 비밀번호 변경 방법

비밀번호는 `js/main.js` 안의 `simpleAccounts` 목록에서 바꿉니다.

예시:

```js
{ id: "gn01", password: "1001", label: "gn01" }
```

이 항목을 아래처럼 바꾸면 됩니다.

```js
{ id: "gn01", password: "2486", label: "gn01" }
```

## 운영 순서

1. 교수님에게 아이디와 비밀번호를 안내합니다.
2. 변경 요청이 오면 `simpleAccounts`에서 해당 계정의 비밀번호를 수정합니다.
3. 변경 후 GitHub에 다시 올리면 실제 홈페이지에 반영됩니다.

## 주의사항

- 이 방식은 간단한 로그인 방식이라 보안이 강하지 않습니다.
- 중요한 개인정보 저장용 서비스에는 적합하지 않습니다.
- 계정 수가 많아지거나 직접 비밀번호 변경 기능이 필요해지면 별도 로그인 시스템으로 전환하는 것이 좋습니다.
