시험용 계정 안내문

대한민국산업현장교수 경남협의회 홈페이지 접속을 위한 시험용 계정 안내문입니다. 아래 아이디와 초기 비밀번호로 로그인한 뒤, 상단의 비밀번호 변경 버튼을 이용해 새 비밀번호로 바꿔 사용할 수 있습니다.

Official Notice

대한민국산업현장교수 경남협의회 공식 배포용 안내문

본 안내문은 경남협의회 홈페이지 시험 운영을 위해 배정된 계정을 전달하는 용도입니다. 전달받은 계정은 본인 확인 후 사용해 주시고, 로그인 이후에는 비밀번호를 변경하여 사용해 주시기 바랍니다.

Caution

주의사항

  • 계정 정보는 본인 외 타인에게 전달하지 않습니다.
  • 첫 로그인 후 반드시 새 비밀번호로 변경해 주세요.
  • 로그인 정보 관련 문의는 관리자에게 직접 확인해 주세요.
  • 비밀번호 변경 내용은 현재 사용한 같은 브라우저에서만 유지될 수 있습니다.
  • 관리자도 변경된 개인 비밀번호를 중앙에서 확인할 수 없습니다.

로그인 방법

  1. 1. 홈페이지 접속
  2. 2. 배정된 아이디 입력
  3. 3. 초기 비밀번호 입력
  4. 4. 로그인 후 상단의 비밀번호 변경 버튼 클릭
  5. 5. 본인만 아는 새 비밀번호로 변경

현재 비밀번호 변경은 같은 PC, 같은 브라우저 기준으로만 유지될 수 있습니다. 다른 브라우저나 다른 기기에서는 초기 비밀번호로 다시 로그인될 수 있습니다.

안내 문구 예시

아래 배정된 아이디와 초기 비밀번호로 먼저 로그인해 주세요. 첫 로그인 후에는 상단의 비밀번호 변경 버튼을 눌러 새 비밀번호로 바꾸어 사용하시면 됩니다.

참고로 변경된 비밀번호는 현재 사용한 브라우저에서만 유지될 수 있으며, 관리자도 변경된 비밀번호를 별도로 확인할 수 없습니다.
