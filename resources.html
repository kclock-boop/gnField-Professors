교육자료, 회의양식, 참고문서, 사례 아카이브를 한곳에서 관리하는 자료실입니다. 로그인한 사용자는 자료를 등록할 수 있고, 등록된 자료는 목록 검색과 분류 기능으로 빠르게 찾을 수 있습니다.

Upload Guide

자료 등록 안내

  • 브라우저 업로드 파일은 현재 사용 중인 브라우저에 저장됩니다.
  • 링크 등록은 웹주소만 입력하면 바로 공개 목록에 반영됩니다.
  • 로그인한 계정만 자료 등록이 가능하며, 등록자는 자신의 자료를 삭제할 수 있습니다.
  • 관리자 계정은 모든 등록 자료를 정리할 수 있습니다.
GitHub Pages는 정적 사이트라 서버 저장 업로드는 바로 지원하지 않습니다. 그래서 현재는 시험 운영용으로 브라우저 저장형 업로드를 적용했습니다.

등록 가능 상태

비로그인

자료 등록은 로그인 후 사용할 수 있습니다. 현재 페이지는 공개로 열리지만, 등록 기능은 로그인한 사용자만 사용할 수 있습니다.

AX교재TF 로그인 관리자 페이지
Resource Upload

자료 등록 폼

제목, 분류, 설명을 입력하고 파일 또는 링크를 등록해 주세요. 파일 업로드와 링크 등록 중 하나만 입력해도 자료를 만들 수 있습니다.

로그인 후 등록 가능

파일 업로드는 현재 브라우저 저장형입니다. 다른 PC나 브라우저에서는 자동 공유되지 않습니다.

등록 규칙

  • 제목은 필수입니다.
  • 링크 또는 파일 중 하나는 반드시 있어야 합니다.
  • 등록자는 로그인한 계정 이름으로 자동 기록됩니다.
